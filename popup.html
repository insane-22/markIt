Instructions

How to Highlight

  1. Select the text you want to highlight.
  2. Right-click the selected text and choose the "Highlight" option from the context menu.
  3. The selected text will be highlighted with the current active color.

How to Add a Note

  1. Hover over your target highlight, you would see an options popup.
  2. Click the "Add Note" button, which is second option.
  3. Type your note and save it.
  4. Your note will appear in the highlights list and can be searched or filtered by date.
